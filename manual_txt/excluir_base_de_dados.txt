====== Excluir Base de Dados ======

Para excluir uma base de dados basta acessar o Menu "Espécime/Base de Dados", localizar a base de dados que deseja excluir, marcar a caixa de seleção e clicar no botão "Excluir", como na imagem abaixo.

{{::excluir_base_de_dados.jpg|}}

Após isso, um alerta de confirmação será exibido com a seguinte mensagem "Confirma a exclusão desta base de dados e todos os seus registros? Obs. Esta ação NÃO poderá ser desfeita.", caso deseje confirmar, basta marcar a caixa de seleção com a mensagem "Excluir todos os registros associados a esta base de dados?" e clicar no botão "Excluir", como na imagem abaixo.

{{::confirmar_excluir_base_dados.jpg|}}

====== Atenção ======

O botão de excluir só será visível para usuários com a permissão de "Excluir espécime", essa opção pode ser solicitada ao Curador/Administrador do Jabot em sua instituição.
 
Para atribuir a permissão de "Excluir espécime" a algum usuário, basta acessar o menu "Admin/Usuário", clicar sobre o nome do usuário que deseja atribuir esta opção, clicar sobre o botão "Permissões" e marcar a opção "Assistente de curadoria – excluir espécime".
Após isso, basta solicitar que o usuário faça um novo login no sistema que a funcionalidade estará disponível.

**Cuidado!** Usuários com essa opção habilitada poderão excluir as bases de dados com todos os registros.
