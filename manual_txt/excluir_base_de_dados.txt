# Excluir base de dados (Jabot)

## Objetivo
Excluir uma base de dados e, opcionalmente, todos os registros associados a ela. Esta ação não pode ser desfeita.

## Como acessar no Jabot
**Espécime > Base de Dados**

## Passos para excluir uma base de dados
1. Acesse **Espécime > Base de Dados**.
2. Localize a base de dados que deseja excluir.
3. Marque a caixa de seleção dessa base.
4. Clique no botão **Excluir**.
5. No alerta de confirmação, leia a mensagem: "Confirma a exclusão desta base de dados e todos os seus registros? Obs. Esta ação NÃO poderá ser desfeita."
6. Se confirmar, marque a caixa "Excluir todos os registros associados a esta base de dados?" e clique em **Excluir**.

## Permissão necessária
O botão Excluir só aparece para usuários com a permissão **Excluir espécime**. Para solicitar: fale com o Curador ou Administrador do Jabot na sua instituição.

Para o administrador atribuir a permissão: **Admin > Usuário** > clique no nome do usuário > **Permissões** > marque **Assistente de curadoria – excluir espécime**. O usuário precisa fazer novo login para a funcionalidade ficar disponível.

## Cuidado
Usuários com essa permissão podem excluir bases de dados com todos os registros. Use com responsabilidade.
