# Consulta interna e gerar etiquetas (Jabot)

## Objetivo
Visualizar todos os itens de uma base de dados pela consulta interna e gerar etiquetas em PDF para os espécimes selecionados.

## Como acessar no Jabot
**Consulta > Consulta**

## Passos para visualizar todos os itens de uma base
1. Acesse **Consulta > Consulta**.
2. No campo **Base de Dados** do formulário de pesquisa, escolha a base desejada.
3. Clique no botão **Consultar**.

## Passos para gerar etiquetas
1. Realize a consulta da base desejada (passos acima).
2. Após exibir os resultados, na paginação escolha quantos itens exibir (10, 20, 50, 100 ou todas) para incluir todos os dados que deseja nas etiquetas.
3. Marque todos os itens usando o primeiro quadradinho da barra antes dos resultados.
4. Clique no botão **Gerar etiquetas para espécimes marcados**.
5. Escolha o tamanho da etiqueta desejada; será aberto um PDF com todas as etiquetas selecionadas.

## Observação importante
Sempre que for necessário gerar mais de 10 etiquetas, o sistema solicita que na opção de ordenação seja selecionado **Emissão de etiqueta (Coletor, número de coleta)**. Assim todas as etiquetas serão geradas seguindo essa ordenação.
