====== Consulta Interna/Gerar etiquetas======

  * Para visualizar todos os itens de sua Base de dados, acesse a consulta interna, no menu Consulta -> Consulta, no campo Base de Dados do formulário de pesquisa, escolha a base desejada e depois clique no botão Consultar.

  * Para gerar as etiquetas, realize a consulta da base desejada(passo 1), após exibir os resultados, observe na paginação se estão sendo mostrados todos os dados que deseja gerar as etiquetas(escolha a opção que satisfaça todos os dados de sua base - 10, 20, 50, 100 ou todas), feito isso, basta escolher a opção de marcar todos os itens(primeiro quadradinho da barra antes dos resultados), agora basta clicar no botão - Gerar etiquetas para espécimes marcados, Escolha o tamanho da etiqueta desejada e irá abrir um PDF com todas as etiquetas selecionadas.
