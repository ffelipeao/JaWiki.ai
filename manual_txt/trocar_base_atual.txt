# Trocar base atual (Jabot)

## Objetivo
Para cadastrar novos registros e trabalhar em uma base de dados específica, ela precisa estar selecionada como base de dados atual. A base atual fica visível logo abaixo do menu.

## Como acessar no Jabot
**Espécime > Trocar base atual**

## Passos para trocar a base atual
1. Clique no menu **Espécime > Trocar base atual**.
2. Selecione a base de dados desejada na lista de bases de dados.
3. Clique em **Salvar**.

A partir daí a base de dados atual ficará sempre visível abaixo do menu, mostrando em qual base você está trabalhando.
