# Base de dados – criar e consultar (Jabot)

## Objetivo
A base de dados é usada quando um coletor volta de uma expedição e quer ter controle de suas coletas. O Jabot permite criar novas bases, consultar bases existentes e alterar informações de uma base.

## Como acessar no Jabot
**Espécime > Base de Dados**

## Consultar bases de dados
Na tela de Base de Dados: digite o nome da base no campo "Base" ou marque o check para listar todas as bases; em seguida clique no botão **Filtrar**. Clicando em uma base da lista, você altera as informações dela.

## Passos para criar uma nova base de dados
1. Acesse **Espécime > Base de Dados**.
2. Clique no botão **Nova base**.
3. Preencha todos os campos necessários. Campos obrigatórios: **Nome da Base**, **Coleção Botânica Padrão** e **Tipo de Coleção**.
4. Clique no botão **Salvar**.

## Observação
Marque a caixa de seleção **Pode Publicar na web** quando quiser que os dados dessa base apareçam na consulta externa (consulta pública).
