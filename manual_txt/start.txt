# Jabot – Manual (índice e visão geral)

## O que é o Jabot
O Jabot é um sistema on-line de gerenciamento de coleções científicas depositadas em herbários, redes laboratoriais de instituições de pesquisas botânicas e coleções vivas de jardins botânicos. Oferece à curadoria, subcuradorias e equipe funções de gerenciamento, acompanhamento de produtividade e relatórios. Permite entrada e disponibilização de dados e imagens dos testemunhos, com análise e melhoria da qualidade dos dados.

## Por onde começar
Consulte **Primeiros passos** no manual para decidir o que deseja fazer (consulta pública, arboreto, etc.).

## Estrutura do manual – principais temas
- **Consultas:** Consulta Pública, Consulta Interna, gerar etiquetas, escolher base de dados, excluir testemunho, coleção correlata, migrar para outra base.
- **Arboreto:** mapa do arboreto, fenologia, controle de foto, manejo (mapa, programação, tipo de atividade, atividade, colaboradores).
- **Espécime:** Registrar Espécime (incluindo imagens sem dados e via coleção correlata), excluir determinação, Coleção Botânica, Base de Dados (criar, excluir), Trocar base atual, Gerenciar usuário da base.
- **Planilha:** Validar planilha, lista de campos básicos, lista de validações, configuração (validação, importador, coordenadas, localidade, logo).
- **Taxonomia:** Gerenciar táxons, táxons adicionados, cadastrar grupo taxonômico, cadastrar táxon.
- **Flora:** Associar/desassociar táxon a base, migrar dados entre espécies.
- **Curadoria:** Coleção correlata, controle de lâminas, exsicata download, guia de remessa, qualidade de dados, unidade geopolítica, unidade de conservação, atualizar consulta pública/imagens, igualar código de barras, documentos.
- **Admin:** Função, log de tentativa de acesso, Perfil, Configuração, Usuário (cadastrar, editar, alterar permissões), Unidade Conservação, Log testemunho.
- **Login:** Login, Esqueci minha senha, Alterar senha.
- **Sobre:** Informações do herbário, adicionar informações, curador, colaboradores.

## Recomendações de uso
Recomenda-se o uso do navegador **Google Chrome** para melhor experiência. Navegadores como Firefox, Opera e Microsoft Edge são testados e costumam funcionar; como estão em constante atualização, não é possível garantir todas as combinações.

## Versão
O manual refere-se ao Jabot versão 4.0 (lançada em 2024, com migração para ambiente de nuvem). Versões anteriores incluíram suporte a exsicatas, arboreto, carpoteca, xiloteca, fototeca e uso por herbários parceiros.
