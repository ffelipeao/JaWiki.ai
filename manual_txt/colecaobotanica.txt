====== Coleção Botânica ======

Para cadastrar/alterar uma Coleção Botânica, Clique no menu **Espécime-Coleção Botânica**, nesse tela será possível ver todas as coleções já cadastradas. 
   - Para alterar basta clicar no nome da coleção que o sistema será redirecionado para a tela de alteração.
  
{{:colecao.jpg?nolink|}}


  - Para cadastrar uma coleção nova basta clicar no botão **Nova Coleção** e será redirecionado para a tela de cadastro.  

Preencha todos os campos, como na figura a seguir. 

{{::cadcolecao.jpg|}}



**Obs. É possível configurar a coleção para seguir o mesmo sequencial de número tombo de uma coleção já cadastrada ou iniciar um novo sequencial.** 

Opções para controle de número de tombo:

   * **Opção 1** - Para utilizar a mesma sequencia de números de uma coleção, basta selecionar na opção 1 a coleção de referencia (principal). Dessa forma, caso a coleção de referência já esteja no número de tombo 100 o próximo número a ser gerado na coleção que está sendo criada, será o 101. O mesmo número será incrementado conforme o cadastro for sendo realizado em casa coleção.
   * **Opção 2** - Para iniciar um novo sequencial, basta informar o número inicial desta sequencia na opção 2. No exemplo da imagem anterior, o primeiro número de tombo a ser criado será o 1000.
