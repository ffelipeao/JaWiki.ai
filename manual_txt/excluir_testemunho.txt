# Excluir testemunho (Jabot)

## Objetivo
Excluir um registro (testemunho) do sistema. A exclusão é definitiva após confirmação.

## Como acessar no Jabot
**Consulta > Consulta** — localize o testemunho; o link para excluir aparece na lista de opções ao clicar no número do tombo.

## Passos para excluir um testemunho
1. Realize a pesquisa na **Consulta > Consulta** e localize o testemunho desejado.
2. Clique no número de tombo para abrir a lista de links/opções.
3. Clique no link **Excluir** para o testemunho escolhido.
4. O sistema redireciona para uma tela de confirmação com os dados do testemunho.
5. Verifique se está tudo correto e confirme a exclusão.

## Permissão necessária
O link para excluir testemunho só aparece para usuários com perfil de **administrador** ou **curador** do sistema.
