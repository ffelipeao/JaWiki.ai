# Função (Jabot)

## Objetivo
O curador pode ver a lista de perfis de usuários, as funções que o sistema oferece e quais usuários estão em cada perfil; além das funções de cada perfil. Útil para atribuir perfis à equipe (permanente ou variável, como estagiários).

## Como acessar no Jabot
**Admin > Função**

## Conteúdo
O sistema possui dezenas de funções (ex.: Adicionar coleção correlata, Adicionar determinação, Administrar Jabot, Alterar Base de Dados, Registrar Espécime, Importar planilha, Qualidade de Dados, etc.). Na tela o curador pode selecionar perfis para a equipe. Para ver as atividades de um perfil: clique no perfil e no botão **Filtrar** (ex.: perfil "Assistente de curadoria").

Exemplo: estagiário pode ter apenas a função de registrar espécimes em uma base específica (Assistente de curadoria - registrar base).
