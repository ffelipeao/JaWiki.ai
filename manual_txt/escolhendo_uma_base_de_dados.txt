# Consultar por base de dados (Jabot)

## Objetivo
Consultar dados no Jabot filtrando pela base de dados. Útil para ver apenas os registros de uma expedição ou projeto específico.

## Como acessar no Jabot
**Consulta > Consulta**

## Passos para consultar por base de dados
1. Efetue login no sistema.
2. Acesse a consulta interna: **Consulta > Consulta**.
3. Na opção "Selecione a base de dados", escolha uma das bases de dados.
4. Clique no botão **Consultar**.

## Observação
Se necessário, é possível combinar o filtro por base de dados com outros filtros, como Família, Gênero e/ou Espécie.
