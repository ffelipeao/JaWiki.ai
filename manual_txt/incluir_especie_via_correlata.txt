# Registrar espécime via coleção correlata (Jabot)

## Objetivo
Registrar um novo espécime usando como base os dados de outro espécime já cadastrado em uma coleção correlata. O formulário é preenchido com os dados do correlato; o usuário ajusta e salva.

## Como acessar no Jabot
**Espécime > Registrar Espécime**

## Passos
1. Selecione a base de dados onde os dados serão cadastrados (Trocar base atual se necessário).
2. Acesse **Espécime > Registrar Espécime** e escolha a coleção botânica que receberá o novo espécime.
3. Na tela do formulário de cadastro, clique no botão **Incluir Dados Via Correlata**.
4. Informe **CódBarras**, **NúmTombo** ou **Coletor e Número de coleta**; selecione a **coleção correlata** e clique em **Buscar**.
5. O espécime encontrado será listado; clique no **CódBarras** para abrir o cadastro com os campos preenchidos com base nesse espécime correlato.
6. Ajuste os dados se necessário e salve o novo registro.
