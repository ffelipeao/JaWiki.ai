====== Registrar Espécime Via Coleção Correlata ======

Para registrar um novo espécime com base nos dados de outro espécime já cadastrado no sistema em uma coleção cujos registros são correlatos.  

Basta seguir os seguintes passos:
Selecione a base de dados correspondente onde os dados serão cadastrados, acesse o menu "Espécime/Registrar Espécime" e escolha a coleção botânica que receberá o novo espécime. Uma tela será aberta com o formulário padrão de preenchimento onde será possível buscar os dados já cadastrados em um espécime de uma coleção correlata. Clique no botão "Incluir Dados Via Correlata" (**Figura 1**), informe o CódBarras, NúmTombo ou Coletor e Número de coleta, também será preciso selecionar a coleção correlata e clicar em "Buscar" para encontrar o espécime na coleção correlata (**Figura 2**). Após isso, esse espécime correlatado será listado para seleção, basta clicar sobre o CódBarras que a tela de cadastro será aberta com os campos preenchidos com base nesse espécime correlato.
