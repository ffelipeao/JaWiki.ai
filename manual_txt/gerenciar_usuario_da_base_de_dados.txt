# Gerenciar usuário da base de dados (Jabot)

## Objetivo
Após criar uma base de dados, associar os usuários que poderão acessá-la e definir quem é responsável pela base. Apenas responsáveis podem associar ou excluir usuários da base.

## Como acessar no Jabot
Troque a base atual para a base desejada (**Espécime > Trocar base atual**). Em seguida clique em **Gerenciar usuário da Base de Dados** (no menu ou na tela da base).

## Passos
1. Selecione a base de dados desejada como base atual (**Espécime > Trocar base atual**).
2. Clique em **Gerenciar usuário da Base de Dados**.
3. Na tela é possível ver quais usuários têm acesso à base e quem é responsável.
4. Para associar: clique em **Associar novos usuários**, selecione o usuário na lista, marque se ele também será responsável pela base e clique em **Salvar**.

Somente os responsáveis pela base podem associar ou excluir usuários nela.
