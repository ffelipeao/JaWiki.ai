# Migrar dados para outra base de dados (Jabot)

## Objetivo
Transferir registros de uma base de dados para outra, seja um registro por vez ou vários em lote. Requer permissão específica.

## Como acessar no Jabot
**Consulta > Consulta** — faça a busca dos registros desejados; nas opções de cada tombo ou na barra de ações em lote aparecerá a opção de migrar.

## Migrar um único registro
1. Faça login e abra a consulta interna: **Consulta > Consulta**.
2. Realize a busca e exiba os resultados.
3. Em cada número de tombo, se você tiver permissão, aparecerá um botão com opções; escolha **Migrar para Outra Base**.
4. Na nova tela: confira o resumo, o campo "Base de dados Atual" e selecione a **Nova base de dados** que receberá o registro.
5. Clique em **Salvar**.

## Migrar vários registros em lote
1. Na consulta interna, realize a busca desejada.
2. Marque os registros que deseja migrar.
3. Clique na opção **Em Lote - Migrar para Outra Base**.
4. Na tela exibida, confira a lista de registros selecionados e escolha a nova base de dados.
5. Marque novamente os registros que deseja migrar (se necessário), escolha a nova base e clique em **Salvar**.

A migração será efetuada para os registros marcados.
