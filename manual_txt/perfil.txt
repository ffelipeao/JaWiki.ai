# Perfil (papel) de usuário (Jabot)

## Objetivo
O sistema possui perfis (papéis) que agrupam funções. O curador pode ver quais perfis cada usuário tem e gerenciar atribuições. Acesso pela página Admin / Papel (sinônimo de Perfil).

## Como acessar no Jabot
**Admin > Papel** (ou Perfil)

Na tela: selecione o perfil e clique em **Filtrar** para ver os usuários daquele perfil; ou consulte pelo nome do usuário para ver os perfis/papéis dele.

## Perfis disponíveis (exemplos)
Administrador Sistema Jabot; Assistente de curadoria (e variações: CV, excluir espécime, gerar duplicatas, registrar base, registrar espécime); Assistente curadoria xiloteca; Assistente de lâminas da xiloteca; Curador imagem; Curador Jabot; Especialista (Flora RJ, detalhar táxon, gerenciar táxon); Importador planilha; Importador planilha para coleções simultâneas; Supervisor Reflora. Cada perfil possui funções associadas definidas na configuração do sistema.
