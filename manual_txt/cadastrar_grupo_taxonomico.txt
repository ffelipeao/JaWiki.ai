# Cadastrar grupo taxonômico (Jabot)

## Objetivo
Informar ou corrigir o grupo taxonômico associado a uma família. O sistema possui rotina noturna que carrega automaticamente o grupo taxonômico para famílias que ainda não têm; esta tela serve para correção ou cadastro manual quando necessário.

## Como acessar no Jabot
**Taxonomia > Cadastrar Grupo Taxonômico**

## Passos
1. Acesse **Taxonomia > Cadastrar Grupo Taxonômico**.
2. Selecione a família desejada.
3. Selecione o grupo correspondente.
4. Clique no botão **Salvar**.
