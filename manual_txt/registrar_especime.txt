# Registrar espécime (Jabot)

## Objetivo
Cadastrar um novo registro de espécime (testemunho) no Jabot. Antes de cadastrar é preciso selecionar uma base de dados que faça parte da coleção desejada.

## Como acessar no Jabot
**Espécime > Trocar Base de Dados** — para selecionar a base antes de registrar.
**Espécime > Registrar Espécime** — para abrir o cadastro.

## Passos para registrar espécime
1. Se necessário, troque a base de dados: **Espécime > Trocar Base de Dados** e selecione a base.
2. Acesse **Espécime > Registrar Espécime**.
3. Escolha a Coleção Botânica para a qual deseja registrar. Se nenhuma coleção aparecer, você não tem permissão; entre em contato com o administrador.
4. Preencha a tela de cadastro. A escolha do tipo de coleção define quais campos aparecem.

## Campos principais da tela de cadastro

### Identificação e coleção
- **Coleção Botânica** — tipo de coleção (definido na tela anterior).
- **Número Tombo** — para espécime já tombado, digite o valor; a caixa "Tombado" vem marcada (dados públicos). Se não informar, o sistema gera novo número.
- **Sufixo** — opcional; usado para testemunhos com mesmo número de tombo e conjunto de exsicatas.

### Dados taxonômicos
- **Cf. / aff. OU !** — detalhe da identificação taxonômica.
- **Táxon** — digite família/gênero; o sistema sugere espécies. Selecione o táxon.
- **Determinado** — nome do determinador.
- **Dia, Mês, Ano** — data da determinação.
- **Notas sobre a determinação** — ex.: Phytotaxa 382(2): 222 (2018).
- **Natureza de typus** — ex.: clonótipo, holótipo (campo dicionarizado).

### Dados da coleta
- **Coletor principal**, **Número da coleta**, **Outros coletores**.
- **Projeto/Expedição**, **Núm. coleta projeto/expedição**.
- **Dia, Mês, Ano Coleta**.
- **Nomes vulgares** — nome popular da espécie na região.

### Localização da coleta
- **Unidade geopolítica** — digite o nome (ex.: município) ou procure na lista à direita. Se não existir, solicite ao Curador; cadastro em **Curadoria > Unidade Geopolítica**.
- **Descrição detalhada da localidade**.
- **Latitude mínima e máxima** — formato Grau-Minuto-Segundo; há conversão a partir de decimal (GPS).
- **Coordenadas inferidas** — para revisão de coordenadas. Consulta de amostras para revisão: **Curadoria > Qualidade de Dados > Coordenadas Geográficas**; escolher coleção e "MUNICÍPIOS CUJAS COORDENADAS NÃO CONFEREM COM A LOCALIDADE INFORMADA".
- **Latitude e longitude em G-M-S** — coordenadas revisadas; conversão de decimal disponível.

### Unidade de conservação e ambiente
- **Unidade de conservação** — digite ou selecione na lista; há botão para cadastrar nova unidade.
- **Elev./Prof. Mínima e Máxima** — não registrar unidade de medida no campo.
- **Unidade de Medida** — lista para elevação/profundidade.
- **Descrição do Ambiente** — apenas ambiente da coleta, não do indivíduo.
- **Ecossistema/Tipo vegetação**.

### Sobre o espécime
- **Desaparecido/Morto** — controle em coleção viva (arboreto).
- **Altura**, **Unid. Medida**, **DAP**, **Fuste**.
- **Descrição do indivíduo**, **Habitat**, **Habitat/Forma de vida**.
- **Usos**, **Uso Específico** (etnobotânica), **Frequência**, **Luminosidade**.
- Caixas: **Estéril, Fértil, Possui flor, Com fruto**, etc.
- **Sigla Coleção de Origem**, **Duplicatas**, **Qtd. em Estoque**.
- **Citações na bibliografia** — ex.: Linnaea 12: 74 (1838).
- **Comentários**.

### Registro de indivíduo do arboreto
Certifique-se de estar na base de dados do tipo coleção botânica (arboreto). Campos específicos: **Seção**, **Canteiro**, **Plantado por**, **Dia/Mês/Ano plantio**, **Latitude**, **Longitude** (decimal), **Plaqueado / Plaquear / Restaurar Placa**, **Nome vulgar placa do indivíduo**, **Distribuição geográfica da placa**.

## Buscar imagens sem dados capturados
Se a coleção tiver imagens ainda sem dados digitados, o sistema exibe o botão "Selecionar Imagem" com o número de arquivos disponíveis. Clique para carregar uma imagem e digitar os dados no formulário (com zoom na etiqueta).
