# Editar dados do usuário (Jabot)

## Objetivo
Alterar dados cadastrais de um usuário, desativar acesso ou trocar a senha (com auxílio do administrador).

## Como acessar no Jabot
**Admin > Usuário**

## Passos para editar um usuário
1. Acesse **Admin > Usuário**.
2. Pesquise o usuário por "nome" ou "login" e clique em **Filtrar**.
3. Clique no nome do usuário na lista.
4. Clique no botão **Editar Usuário**.
5. Na tela de edição, faça as alterações necessárias e clique em **Salvar**.

## Outras opções na tela de edição
- **Desativar** — remove o acesso do usuário que não utilizará mais o sistema.
- **Trocar Senha** — opção para o administrador auxiliar o usuário a redefinir a senha.

## Recadastramento
Em certas situações o sistema exibe a tela de Recadastramento de Usuário (por exemplo após um ano de cadastro); siga as instruções na tela para concluir.
