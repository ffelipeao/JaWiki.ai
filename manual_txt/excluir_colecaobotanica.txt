# Excluir coleção botânica (Jabot)

## Objetivo
Excluir uma coleção botânica do sistema. Por segurança, o Jabot não permite excluir coleções que ainda possuam registros associados; é necessário excluir ou migrar os registros antes.

## Como acessar no Jabot
**Espécime > Coleção Botânica**

## Passos para excluir uma coleção
1. Acesse **Espécime > Coleção Botânica**.
2. Localize a coleção que deseja excluir.
3. Marque a caixa de seleção dessa coleção.
4. Clique no botão **Excluir**.
5. No alerta "Deseja excluir a(s) coleção(ões) botânica(s) selecionada(s)?", confirme clicando em **Excluir**.

Esta ação **não** pode ser desfeita.

## Antes de excluir
Se a coleção tiver registros, exclua-os ou migre-os para outra coleção antes. Consulte no manual: **Excluir testemunho** e **Migrar para outra base de dados**.

## Permissão necessária
O botão Excluir só é exibido para usuários com a permissão **Excluir espécime**. Solicite ao Curador ou Administrador. Para atribuir: **Admin > Usuário** > nome do usuário > **Permissões** > marque **Assistente de curadoria – excluir espécime**. O usuário deve fazer novo login.

**Cuidado:** usuários com essa permissão podem excluir coleções, bases de dados e registros.
