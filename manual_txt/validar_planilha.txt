# Validar planilha (Jabot)

## Objetivo
Enviar uma planilha para validação antes da importação. O sistema verifica erros e alertas; quando não houver erros, o botão Importar é exibido para submeter os dados ao banco.

## Como acessar no Jabot
**Espécime > Validar Planilha**

## Passos para validar uma planilha
1. Selecione a coleção botânica.
2. Informe o nome da base de dados que deseja criar ou marque a opção para o nome ser gerado automaticamente.
3. Selecione a planilha no seu computador (navegue pelas pastas).
4. Clique no botão **Enviar** para iniciar a validação.

## Se houver erros ou alertas
O sistema exibe uma tela indicando os tipos de erro. Corrija a planilha no seu computador e envie novamente para validação.
- **X** — erro que precisa ser corrigido.
- **!** — alerta (ex.: diferença em relação à lista do Brasil); não impede a submissão, mas deve ser revisado.

## Finalizar processo
Quando nenhum erro for encontrado, será exibido o botão **Importar**. Clique para submeter os dados validados à importação.

## Observação – nome da base automático
O padrão de nome automático é "Coletas + seu login + data atual". Se importar mais de uma planilha no mesmo dia com nome automático, o sistema pode informar que o nome já existe; nesse caso, informe um nome de base diferente e tente novamente.

## Configuração e formato
- A planilha deve estar no formato .xls e as colunas devem seguir a planilha padrão (baixe a planilha padrão na tela de Validar Planilha).
- Padrão taxonômico: Lista do Brasil. Coordenadas: use vírgula, não ponto. Retire espaços no nome de um mesmo autor.
- Para configurações da instituição: **Configuração para importação de planilha** no manual (Admin/Configuração).

## Validação para Arboreto (coleção viva)
Na tela de validação, baixe a planilha padrão da coleção viva, selecione uma coleção do tipo Arboreto (ex.: RBv), escolha o arquivo preenchido no novo formato e clique em **Enviar**.

## Importação simultânea para coleções correlatas
Se a função estiver ativada: em **Admin > Configuração**, em "Configurações para importação de planilha", ative "Ativar dupla importação de dados com associação de correlatas" e salve. O usuário precisa da permissão "Importador planilhas para coleções simultâneas" (Admin > Usuário > Permissões). Em **Espécime > Validar Planilha** aparecerá a seleção de coleção principal e correlata. A coleção com planilha mais completa deve ser a principal (ex.: RBv principal e RB correlata). Ao final da validação sem erros, use **Importar**; os dados serão importados em ambas as coleções com associação dos correlatos.
